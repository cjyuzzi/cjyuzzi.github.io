簡介

  • 靜態網站的首選。
  • 客製化的主題：擁有許多免費主題供大家使用，並且能透過修改主題、加入套件來客製化自己的主題。
  • 支援使用 Markdown 語法撰寫。
  • 使用 GitHub Pages 來架設 Hexo 部落格：
    • 免費
    • 穩定 & 安全
    • 大小夠用：大小不能超過 1 GB
    • 流量夠用：每月流量不能超過 100 GB
  • 只能透過 git push 更改部落格樣式、發表或修改文章。
  • 設定部落格相關資訊的專案並不會 push 上版控，因此要保存好。

建立 Github Pages 環境

  • 註冊 Github 帳號
  • 建立 Repository : username.github.io
  • 設定 Repository : settings > Theme Chooser > Choose a theme
  • 準備就緒 : 已經可以瀏覽 https://username.github.io/.

安裝管理文章的後台插件

  • 提供本機測試：架設一個本地伺服器，可以在進行部屬前先查看部落格，例如檢查修改樣板過後呈現的效果。
  • 重點它提供了一個後台讓你管理文章：例如新增、修改或刪除文章等功能。
  • 安裝 : npm i hexo-admin --save
  • 進入後台 : hexo s
