步驟

  • 開啟本機 Hexo 伺服器 : hexo s。
  • 進入後台 : http://localhost:4000/admin。
  • ➕ New Post : 輸入該文章的連結網址名稱 🔗。
  • ✏ 修改文章標題名稱 : 取代預設的連結網址名稱。
  • 🏷 加入 Tags & Categories。
  • 📑 填入使用 Markdown 撰寫的內文 : 可使用自己的 Markdown 編輯器撰寫。語法參考。
  • (選用) 加入 Read More 截斷文章：<!--more-->
  • 停止本機伺服器 : Ctrl+C。
